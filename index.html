Transformers - Animation

CS475 | Computer Graphics Assignment 2.3

Work Done

  • Implemented keyframe string generation and storing to file keyframes.txt
  • Reading back of keyframes from file line by line
  • Set attribute targets and speeds for correct linear interpolation
  • Invoking interpolation function in main loop, saving buffer GL_BACK as image each time the loop runs
  • Stringing the dumped image files into a video using libav
  • Adding audio and uploading to YouTube

Our flying transformer doing a series of aerobatics followed by a dive to the ground, ending in an untransformation (back into humanoid form).
